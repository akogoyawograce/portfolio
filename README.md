# Portfolio — Akogo Yawo Grace

Portfolio personnel developpe en HTML, CSS et JavaScript purs. Design responsive, mode sombre, animations au scroll.

## Apercu

- Hero avec animation de typing
- Sections : A propos, Competences, Projets, Formation, Contact
- Geolocalisation fixe (Lome, Togo) liee vers Google Maps
- Liens WhatsApp et appel direct
- Mode sombre/clair avec persistance localStorage
- Menu hamburger mobile

## Technologies

- HTML5, CSS3, JavaScript (vanilla)
- Google Fonts (Inter)
- Font Awesome 6

## Structure

```
portfolio/
  index.html   — structure et scripts
  index.css    — styles
```

## Deploiement

Compatible GitHub Pages — branche `main`, dossier racine.
